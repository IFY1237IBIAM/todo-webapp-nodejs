# To-Do List Web App

## Overview
This is a browser-based To-Do List Web App built with HTML, CSS, and JavaScript. Users can add, delete, and mark tasks as complete. Tasks are saved in the browser's local storage so they persist across sessions.

## Features
- Add new tasks
- Mark tasks as complete
- Delete tasks
- Tasks saved in Local Storage
- Two HTML pages: `index.html` (main app) and `about.html` (instructions & overview)
- Fully interactive
