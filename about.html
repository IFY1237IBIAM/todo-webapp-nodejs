App Overview

This To-Do List Web App allows users to add, delete, and mark tasks as complete. Tasks are saved in the browser's local storage.

Instructions

  1. Go to the main To-Do List page.
  2. Add tasks using the input box and click "Add Task".
  3. Mark tasks as complete by clicking the checkbox.
  4. Delete tasks by clicking the "Delete" button.
